# Транспорт Вінниця - версія 3

## Опис
Повністю незалежна версія PWA додатку для оплати громадського транспорту у Вінниці.

## Структура
- `transport.html` - головна сторінка (стартова точка)
- `index.html` - архів квитків
- `payment.html` - оплата проїзду
- `qr.html` - сканер QR-кодів
- `settings.html` - налаштування додатку
- `index.js` - вся JavaScript логіка
- `manifest.json` - PWA маніфест
- `service-worker.js` - офлайн підтримка
- `icons/` - іконки для PWA

## Особливості v3
- Повністю незалежна від інших версій
- Всі файли самодостатні
- Оптимізовані шляхи для GitHub Pages
- PWA підтримка з офлайн режимом
- QR сканування через камеру
- Статистика поїздок
- Fullscreen режим

## Запуск
Відкрийте файл `transport.html` у браузері або розгорніть на веб-сервері.

Для GitHub Pages: `https://rotor003.github.io/prvtfreebus/v3/transport.html`
